
## 📊 Supply Chain Dashboard – Supplier Summary

### 🔍 Overview

This project presents an interactive Excel dashboard designed to analyze supplier performance and overall supply chain efficiency. The dashboard enables stakeholders to monitor key metrics such as revenue contribution, defect rates, operational costs, and delivery timelines.

### 🎯 Objectives

* Evaluate supplier performance across multiple dimensions
* Identify high-risk suppliers based on defect rates and delays
* Analyze cost efficiency and revenue contribution
* Support data-driven decision-making in supply chain management

### 📈 Key Features

* Dynamic dashboard using Pivot Tables and Slicers
* KPI tracking (Total Revenue, Total Orders, Avg Cost, Shipping Cost)
* Supplier-wise and product-wise performance analysis
* Visual insights using bar charts and distribution charts
* Automated data cleaning and refresh using VBA

### 🧠 Key Business Insights

* Supplier 1 contributes the highest share of total supply but also has the highest defect rate, indicating a quality-risk trade-off.
* Skincare products generate the highest revenue, making them the primary business driver.
* Supplier 4 shows higher manufacturing costs without proportional output, suggesting inefficiency.
* Supplier 3 has longer lead and manufacturing times, potentially impacting delivery performance.
* Supply dependency is concentrated among top suppliers, posing a potential risk to operations.

### 🛠️ Tools & Technologies

* Microsoft Excel (Advanced)
* Pivot Tables & Charts
* Data Validation & Slicers
* VBA (for automation)
  
### ⚙️ Automation

* Automated data cleaning (handling missing values)
* One-click dashboard refresh using VBA
* Event-based refresh on workbook open

### 📌 Use Case

This dashboard can be used by operations and supply chain teams to monitor supplier performance, identify inefficiencies, and improve overall decision-making.

### 🚀 Future Improvements

* Integration with SQL database for real-time data
* Migration to Power BI for advanced visualization
* Predictive analytics for demand and defect forecasting
